What is InfoLister?

InfoLister is an extension for Mozilla Firefox and Thunderbird, that collects various information about the host application, optionally saving it to a file or uploading it via FTP to your site. The information includes the list of installed extensions, themes, and plugins.

Install InfoLister 0.10.3 for Firefox 3.6 and Thunderbird 3.

Other versions:

  • InfoLister 0.10.4 alpha 1 for Firefox 4, Thunderbird 3.1 and 3.3. (Leave feedback on mozillazine or Google Code.)
  • InfoLister 0.10.1 for Firefox 2, 3, 3.5, Thunderbird 1.5 and 2, Sunbird 0.7 and later, SeaMonkey 2 alphas.

Also available from addons.mozilla.org. You can read the changelog to see the list of changes between the versions.

After installation, access InfoLister via the Tools menu.

Features:

  • Lists installed extensions, themes, and plugins.
  • HTML, plain text, BBCode, XML, and custom output.
  • FTP upload (help).
  • Quick access: menu item (Tools > InfoLister), toolbar button, or about:info in the Location Bar.

Example output: HTML, plain text, XML (styled with XSLT+CSS).

Sample XSLT transforms: to XHTML, to text, to BBCode.
